# Guia Passo a Passo: Criando Links de Assinatura no Asaas (Sandbox)

Siga este guia para criar os links de pagamento para os seus planos no ambiente de testes do Asaas.

## 1. Acesso ao Sandbox
1. Acesse: **[https://sandbox.asaas.com/](https://sandbox.asaas.com/)**
2. Faça login com sua conta de testes (ou crie uma se não tiver).

## 2. Criar uma Assinatura (Plano)
No menu lateral esquerdo, clique em **Assinaturas** (ícone de calendário/recorrência).

1. Clique no botão **"Nova assinatura"**.
2. **Cliente:** Se não tiver um cliente de teste, crie um novo (ex: "Cliente Teste").
3. **Valor:** Insira o valor do plano (ex: `69,90`).
4. **Periodicidade:** Selecione **Mensal**.
5. **Forma de pagamento:** Marque **Boleto, Pix e Cartão** (para dar todas as opções).
6. **Descrição:** Coloque o nome do plano (ex: "Jhonjhon Club Corte").
7. Clique em **"Salvar"** ou **"Criar assinatura"**.

> **Nota:** O Asaas foca muito em criar assinaturas *para um cliente específico*. Para criar um **Link Geral** que qualquer pessoa possa assinar, o processo é via **Link de Pagamento**. Veja abaixo.

---

## 3. (Recomendado) Criar Link de Pagamento Recorrente
Para colocar no site, o ideal é um **Link de Pagamento** para criar assinaturas.

1. No menu lateral, vá em **Cobranças** -> **Links de pagamento**.
2. Clique em **"Criar link de pagamento"**.
3. **O que você quer vender?**: Selecione **"Assinatura"** (Plano).
4. **Nome do Link:** Coloque o nome do plano (ex: `Jhonjhon Club Corte`).
5. **Valor:** `69,90`.
6. **Periodicidade:** Mensal.
7. **Opções:**
   - Marque "Cartão de Crédito" e "Pix".
   - Você pode pedir endereço se quiser, mas geralmente não precisa para barbearia.
8. Clique em **"Salvar"** ou **"Gerar Link"**.

## 4. Copiar o Link
1. Após criar, o link vai aparecer na lista.
2. Clique no ícone de **Copiar** ou abra o link para testar.
3. O link deve se parecer com: `https://sandbox.asaas.com/c/1234abcd...`

## 5. Repita para os Outros Planos
Faça o mesmo processo "Passo 3" para os outros planos:

- **Barba:** R$ 75,00
